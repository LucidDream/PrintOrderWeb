Sample PDF Files for Testing
============================

This folder contains sample PDF files for testing the Print Order Web application.

Included Files:
- sample_test_page.pdf - A simple single-page A4 test document

How to Use:
1. Start the Print Order Web application
2. Navigate to http://127.0.0.1:5000
3. Upload one of these sample PDF files
4. Configure order settings
5. Submit the order

You can also add your own PDF files to this folder for testing.
